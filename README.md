# Zen Spa

See the site live: https://web301-assignment-2.vercel.app

This is a site for a spa that offers packages and products. The home page welcomes you and directs you to the pages for the packages and prodcuts. For the packages, the user can click on the package to purchase it. There is no tax. For the products, the user can click on the product to purchase it. There is a sales tax for this. The user can also book an appointment by filling out a form on the Book Appointment page. Once the form is filled out, an alert with all the details of the form is displayed.
